أدخل رابط Google Apps Script للبدء

📋 خطوات الإعداد:

  1. افتح Google Sheets وأنشئ جدولاً جديداً
  2. اذهب إلى Extensions → Apps Script
  3. الصق محتوى ملف Code.gs
  4. اضغط Deploy → New Deployment → Web App
  5. اضبط "Execute as: Me" و "Who has access: Anyone"
  6. انسخ رابط الـ Web App والصقه هنا
